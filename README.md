BlackEye Emulator
=================
The best DarkOrbit private server.

Emulator Structure
------------------
The emulator is a *Maven* project made with *IntelliJ IDEA 14*.

The source files are under *src/main/java* and are under the package *com.manulaiko.balckeye*. The main class is *com.manulaiko.blackeye.launcher.Main* and sets everything for starting the servers.

Libraries Used
--------------
The libraries used are (maven coordinates):
 * org.json:json:2.0
 * mysql:mysql-connetor:5.1.38

The *.jar* files are located in *lib*.
